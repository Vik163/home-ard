# Print

## Printf()

[Сайт на тему](https://codelessons.dev/ru/printf-in-c-cplusplus/)

`printf("%s%.1f %s\n", title, value, item);`

- `%` - разделяет значения
- `\n` - перенос на другую строку
- `s` - строка (`const char *name`)
- `.1f` - число с запятой (`float value`) цифра `1` - число символов после запятой
